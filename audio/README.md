# Dossier audio

Dépose ici tes fichiers `.mp3` en respectant exactement ces noms (voir le README principal à la racine du dépôt pour le détail) :

**Cours d'arabe :** `sheet1.mp3` à `sheet13.mp3`
**Vocabulaire infirmier :** `nurse1.mp3` à `nurse6.mp3`

Dès qu'un fichier portant le bon nom est présent, le lecteur audio apparaît automatiquement sur la page du cours correspondant — pas besoin de modifier le code.

Ce fichier (`README.md`) ne gêne rien : tu peux laisser le dossier comme ça même vide, GitHub a juste besoin d'au moins un fichier pour "voir" un dossier vide.
